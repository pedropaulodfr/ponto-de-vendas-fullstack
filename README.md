# Cadastro Ponto de Vendas
## _Teste full-stack_


## Bibliotecas utilizadas

- Axios
``` sh
npm i axios
````
- Leaflet
``` sh
npm install react react-dom leaflet react-leaflet
```
- React-Icons
``` sh
npm install react-icons --save
```
- React-Moment
``` sh
npm i react-moment
```
- React-Router-Dom
``` sh
npm i react-router-dom
```

## Como Rodar
Digitar no terminal:
``` sh
npm start
```
